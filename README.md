# Welcome to the C Time Suite!
#### This is a Windows console time utility written in C that features the following:
- Digital Clock
- Stopwatch
- Timer

## Features

### Digital Clock
Displays the current local system time.
- B: Back to Menu

### Stopwatch
Stopwatch with lap functionality.
- S: Start
- R: Stop/Resume
- Space: Lap/Reset
- B: Back to Menu

### Timer
Timer that supports these input formats: SS, MM:SS, and HH:MM:SS.
- S: Start
- R: Stop/Resume
- Space: Cancel
- ESC: Back to Menu
